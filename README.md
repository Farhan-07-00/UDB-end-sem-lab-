# Driver Location Visualization using Streamlit and MongoDB

This project demonstrates a Streamlit dashboard that visualizes driver locations on a map using data stored in MongoDB.  
It also demonstrates MongoDB indexing, query optimization, and covered queries using PyMongo.

This project was developed as part of the **Unstructured Database Lab (UDB)**.

---

## Features

- Visualize driver locations on a map using Streamlit
- Store and retrieve driver data using MongoDB
- Create compound and single-field indexes
- Perform optimized queries using PyMongo
- Analyze query execution plans using `explain()`
- Implement covered indexes for efficient query performance

---

## Technologies Used

- Python
- Streamlit
- MongoDB
- PyMongo
